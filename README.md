# swol-app
You will need to create a Firebase app and paste the configuration in an .env files, like this:
VUE_APP_API_KEY=
VUE_APP_DOMAIN=
VUE_APP_DATABASE=
VUE_APP_PROJECT_ID=

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
